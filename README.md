FAST-CAMPUS_WEB
===================

#### :dart: Learning Goal
- 컴퓨터 역사 및 기본 구조 이론지식 이해
- 컴퓨터 및 프로그래밍 관련 실무용어 습득
- Python, html, css, javascritp 문법 이해 및 Django, Django Rest API 활용
- Django Application 개발(Team project)
